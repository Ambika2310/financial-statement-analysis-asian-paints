# Financial Statement Analysis – Asian Paints

This project presents a comprehensive Financial Statement Analysis (FSA) of Asian Paints using publicly available financial data.

## Project Components
- Income Statement Analysis
- Balance Sheet Analysis
- Cash Flow Statement Overview
- Common Size Analysis
- Ratio Analysis
- Trend Analysis

## Company Analyzed
Asian Paints Ltd.

## Tools Used
- Microsoft Excel
- Financial modeling techniques
- Data sourced from Annual Reports and Investor Relations materials.

## Key Insights
- Strong operating cash flow generation
- Conservative capital structure
- Stable profitability with fluctuations due to raw material costs
- Asset expansion supporting business growth

## Files Included
- `FA_Capstone_Project.xlsx` – Financial model and calculations

## Data Sources
- Asian Paints Annual Reports
- Investor Presentations
- Company Website
